# AI Brief Daily (AI简报每日速递)

> 每日早晨7点自动更新，AI、国际、财经、大模型领域核心要闻速递

## 功能特性

- 🎨 现代极简设计，响应式布局
- 🤖 每日北京时间早上7点自动更新
- 📰 四大分类：AI要闻、国际速览、财经脉动、大模型洞察
- 📱 完美支持手机端浏览

## 本地预览

```bash
open index.html
# 或
python3 -m http.server 8080
```

## 项目结构

```
project-x/
├── index.html          # 网站主页面
├── data/
│   └── news.json       # 新闻数据
├── scripts/            # 更新脚本
├── archives/           # 历史归档
└── logs/               # 更新日志
```
